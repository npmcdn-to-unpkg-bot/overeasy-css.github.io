Border radiuses

.border.rounded
.border.rounder

Note: .unrounded would be used to escape opinionated styling, such as in buttons or input fields. Elements are not typically rounded by default.

Colors

Border types and colors are declared separately - the color utility will adjust the color of the set border type.

These utility names and values are auto-generated from the color palette.

Base Colors

.border-gray

Hover Colors

To achieve a border that only appears on hover, there is an additional .border-transparent utility. This acts like a color, so there is actually a border there (so spacing remains consistent), it just only becomes visible on hover.

.hover-border-red
.hover-border-yellow
.hover-border-green
.hover-border-blue
.hover-border-gray
<div class="border border-transparent hover-border-red">.hover-border-red</div>
<div class="border border-transparent hover-border-yellow">.hover-border-yellow</div>
<div class="border border-transparent hover-border-green">.hover-border-green</div>
<div class="border border-transparent hover-border-blue">.hover-border-blue</div>
<div class="border border-transparent hover-border-gray">.hover-border-gray</div>
