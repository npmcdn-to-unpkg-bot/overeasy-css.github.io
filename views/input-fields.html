Input Fields

Text fields

Use .label for labels and .field for text inputs

<div class="mb1">
  <label for="input1" class="label">Label1<label>
  <input type="text" id="input1" class="field">
<div>

<div class="mb1">
  <div class="flex flex-center">
    <label for="input2" class="label">Label2<label>
    <input type="text" id="input2" class="field">
  <div>
<div>

Textarea fields

Use the same .field class as text inputs

<div class="mb1">
  <label for="area1" class="label">Label1<label>
  <textarea type="text" id="area1" class="field"></textarea>
<div>

<div class="mb1">
  <div class="flex">
    <label for="area2" class="label">Label2<label>
    <textarea type="text" id="area2" class="field"></textarea>
  <div>
<div>
