Getting Started

OverEasy is a collection of targeted utilities packaged as individual Node modules. You can install and use them individually if you prefer, but the simplest way to get started is to just install the main package:

npm install overeasy

This package will install all the individual module packages automatically as dependencies. If you wanted to customize your build, this will have everything you need to do it, but it also contains pre-compiled CSS, which you can just link to your page to get started right away. No assembly required.

<link rel="stylesheet" href="node_modules/overeasy/css/overeasy.css"> // Expanded ~22 KB
<link rel="stylesheet" href="node_modules/overeasy/css/overeasy.min.css"> // Minified ~18 KB

overeasy-config

Most of OverEasy is in the form of inline utilities which are immutable, but there are also some opinions baked into the framework for rapid deployment purposes. This module is where variable opinions for all the other modules are set, so if you want to change a setting you don't need to hunt through everything to find it.
