OverEasy

A deliciously complete OOCSS grid and style framework
GitHub

OverEasy is a modular CSS toolkit for rapid and flexible development

Object Oriented

OverEasy is a library of atomic CSS style utilities to mix and match however your view requires

Immutable

Classes are based on their function, so they are interchangable and independent of context

Customizable

All opinionated settings configurable from one place, everything from the color palette to the number of columns in the grid

Lightweight

A complete style and layout framework for less than 20 KB file size

Intuitive

The classes are semantically named and titled relevant to their function, so they are natural to compose

Responsive

Mobile-first grid orientation, size-responsive utilities let you design views that work seamlessly on any display
